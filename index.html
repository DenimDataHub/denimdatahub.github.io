Denim Data Hub

Open Source Data Hub

It is Free

Data hub is built on free components and can be run on free plans of Microsoft Azure. It is free and it will stay free.

It is Open Source

First Development is funded by IoT Rapid Protolabs and European Union's Erasmus+ together with Houston Inc.

Designed for personal and production use

Goal was to create data platform where anyone, from student to professional can start developing data and IoT based solutions with ease.

Testimonial

Our target is to remove the gap from school or home projects to industry level solutions. The tools that one uses for hobby projects or in educational entities shouldn't differ from the ones used in commercial production.

We wanted share our wisdom and created a data hub that can be used by individuals and educational entities and can be extended to high demand industry solutions.

Denim Data Hub is a setup of Microsoft Azure services that together are proven to work and a good setup that scales. It eases the data ingestion from IoT devices.
