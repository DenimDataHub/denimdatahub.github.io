Open Source Data Hub

It is Free

Data hub is built on free components and can be run on free plans of Microsoft Azure. It is free and it will stay free.

It is Open Source

First Development is funded by IoT Rapid Protolabs and European Union's Erasmus+ together with Houston Inc.

Designed for personal and production use

Goal was to create data platform where anyone, from student to professional can start developing data and IoT based solutions with ease.
